**Учебный проект в рамках курса Java QA Engineer. Professional.**

**Домашнее задание ДЗ #2: Реализовать BDD подход**  

**Для запуска проекта необходиимо:**  
Java 20+  
Браузеры Chrome и Firefox  
выбор браузера производиться через изменение systemProperty.browser

**Цель:**  
Реализовать BDD-фичи с использованием cucumber'a.

Необходимо реализовать следующие сценарии при помощи Selenium WebDriver:

**Сценарий 1**.  
Поиск указанного курса (название курса задается в фиче) и его выбор (в случае если несколько, то выбирается случайный).

**Сценарий 2.**  
Поиск курсов, стартующих в указанную дату или позже указанной даты, и вывод информации о них в консоль (название, дата старта).  

**Сценарий 3.**  
Перейти в раздел «Курсы» ➡️ «Подготовительные курсы», выбрать самый дорогой и самый дешевый курс при помощи filter и вывод информации о нем в консоль.
